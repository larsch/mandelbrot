# Mandelbrot Explorer

Fast C++/SDL2 mandelbrot explorer.

Features:

* Interactive
* Multi-threaded
* Zoom/pan, even before current render is complete

## Keyboard navigation

* **Arrow keys:** Pan
* **[** / **]**: Zoom
* **0-9**: Load bookmark
* **Ctrl+0-9**: Save bookmark
* **Shift+1-4**: Change floating point precision (32, 64, 80, 128 bits)
